# extract-metadata

문서에서 메타데이터를 추출하는 스킬

## 트리거

- 파일 분석 요청 시
- 메타데이터 추출 요청 시

## 입력

- `file_content`: 파일 내용 (텍스트)
- `filename`: 파일명

## 출력 형식

반드시 다음 JSON 형식으로만 응답 (다른 텍스트 없이):

```json
{
  "summary": "문서 요약 (2-3문장)",
  "categories": ["카테고리1", "카테고리2"],
  "tags": ["태그1", "태그2", "태그3"]
}
```

## 처리 단계

1. **파일 타입 식별**: 확장자와 내용으로 파일 타입 파악
2. **구조 분석**: 제목, 섹션, 핵심 내용 파악
3. **요약 생성**: 핵심 내용을 2-3문장으로 요약
4. **카테고리 분류**: 규칙에 따라 적절한 카테고리 선택
5. **태그 추출**: 핵심 키워드 3-7개 추출
6. **JSON 출력**: 정해진 형식으로 결과 반환

## 주의사항

- JSON 외 다른 텍스트 출력 금지
- 마크다운 코드 블록 없이 순수 JSON만 출력
- 빈 필드 없이 모든 필드 포함
- tags는 최소 3개 이상
